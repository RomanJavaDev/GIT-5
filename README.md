# GIT-5

1. В статическом блоке инициализирован словарь map парами [число-слово] от 0 до 12 включительно.
Например, 0 - "ноль", 1 - "один", 2 - "два"
2. Нужно считать с консоли имя файла, считается содержимое файла.
3. Заменятся все числа на слова при помощи словаря map.
4. Результат будет выведен на экран.


Пример данных в файле:
Это стоит 1 бакс, а вот это - 12 .
Переменная имеет имя file1.
110 - это число.

Пример вывода в консоль:
Это стоит один бакс, а вот это - двенадцать .
Переменная имеет имя file1.
110 - это число.
